# Virtua Pay - Frontend

A React + Tailwind CSS frontend for the Virtua_PAY_ALTERNATIVE system.

## Setup

1.  Navigate to the `frontend` directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```

## Features

- **Products**: Browse available premium gadgets.
- **Purchase**: Integrated with multiple payment gateways (Razorpay, Cashfree).
- **History**: View past transactions and their statuses.
- **Settings**: Toggle the active payment gateway dynamically.
